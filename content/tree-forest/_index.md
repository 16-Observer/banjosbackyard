Welcome to the Tree Forest — a living list of trees currently growing on the homestead. Add a new tree by creating a markdown file in content/tree-forest with front matter (name, species, planted, notes).

Example entries below — edit or add your own:

- Apple (Malus domestica) — planted 2019 — producing small apples, espaliered near the north fence.
- Plum (Prunus domestica) — planted 2021 — young tree, not yet fruiting.

To add a tree, create a file like content/tree-forest/my-apple.md or run `hugo new tree-forest/my-apple.md`:

---
title: "My Apple"
species: "Malus domestica"
planted: "2019"
notes: "Espaliered, semi-dwarf"
---

A short description here.
